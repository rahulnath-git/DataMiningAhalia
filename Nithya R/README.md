# Data Preprocessing and Decision Tree Classification

## Objective
To implement data preprocessing, feature selection, decision tree classification, and evaluation using Python.

## Dataset
- 100 student records
- Attributes: Age, Study_Hours, Attendance, Internal_Marks, Result

## Steps Implemented
1. Data loading and exploration
2. Handling missing values
3. Normalization
4. Label encoding
5. Feature selection using Gini Index
6. Decision Tree classification
7. Model evaluation

## Tools Used
- Python
- Pandas
- Scikit-learn
- VS Code
