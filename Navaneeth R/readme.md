# Data Preprocessing and Decision Tree Classification

## Assignment Information
- Course: Data Mining
- Assignment: 1
- Topic: Data Preprocessing and Decision Tree Classification

## Objective
To implement a complete data mining pipeline including:
- Data loading and exploration
- Data preprocessing
- Feature selection using Gini Index
- Decision Tree classification
- Model evaluation

## Dataset Description
The dataset contains 100 student records with the following attributes:
- Age
- StudyHours
- AttendancePercent
- InternalScore
- Result (Pass / Fail)

## Tools and Libraries
- Python
- Pandas
- NumPy
- Scikit-learn

## Steps Performed
1. Dataset loading and analysis
2. Encoding and normalization
3. Feature importance calculation
4. Decision tree model training
5. Performance evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
